# Owner Daily Dashboard Guide

**One-Page Daily Routine**  
**Time Required**: 25 minutes total  
**Purpose**: Run business daily without technical help

---

## Morning Routine (10-15 minutes)

### Step 1: Open Dashboard (2 minutes)
1. Login to ERP system
2. Go to **Dashboard** (home screen)
3. Check today's date is correct

**✅ What You Should See**:
- Sales summary (today and yesterday)
- Production overview (pending steps)
- Cash/payment status

---

### Step 2: Check Sales Numbers (3 minutes)
**Look at these numbers**:
- **Today's Sales Count**: How many sales today?
- **Today's Sales Amount**: Total money from sales
- **Yesterday's Sales**: Compare with today

**✅ Good Signs**:
- Sales are being created
- Amounts look normal
- No sudden drops

**⚠️ Warning Signs**:
- Zero sales (check if users are working)
- Very low sales (check if system is being used)
- Sudden drop (check for problems)

---

### Step 3: Check Production Status (3 minutes)
**Look at these numbers**:
- **Dyeing Count**: How many dyeing steps pending?
- **Handwork Count**: How many handwork steps pending?
- **Stitching Count**: How many stitching steps pending?
- **Completed Count**: How many steps completed?

**✅ Good Signs**:
- Steps are being updated
- Completed count is increasing
- No steps stuck for days

**⚠️ Warning Signs**:
- Same steps pending for 3+ days (workers not updating)
- Zero updates (workers not using system)
- High pending count (production bottleneck)

---

### Step 4: Check Cash/Payments (2 minutes)
**Look at these numbers**:
- **Paid Sales**: How many sales are paid?
- **Due Sales**: How many sales are unpaid?
- **Total Due Amount**: How much money is pending?

**✅ Good Signs**:
- Most sales are paid
- Due amount is reasonable
- Payments are being recorded

**⚠️ Warning Signs**:
- Too many unpaid sales (check payment recording)
- High due amount (check collection)
- No payments recorded (check if system is being used)

---

### Step 5: Check Alerts/Warnings (2 minutes)
**Look for**:
- Red error messages
- Yellow warning messages
- System notifications

**✅ Good Signs**:
- No errors or warnings
- System running smoothly

**⚠️ Warning Signs**:
- Error messages (contact support)
- Warning messages (check what's wrong)
- System notifications (read and act)

---

## Mid-Day Quick Check (5 minutes)

### What to Verify (12:00 PM - 1:00 PM)
1. **Sales are being created**
   - Check if new sales appeared since morning
   - If no new sales, check if sales staff is working

2. **Production steps are being updated**
   - Check if step counts changed
   - If no changes, check if workers are updating

3. **No errors reported**
   - Ask team if anyone faced problems
   - If problems, note them for end-of-day review

**✅ Good Signs**:
- System is being used
- Team is working normally
- No complaints

**⚠️ Warning Signs**:
- No activity (team not using system)
- Complaints (address immediately)
- Errors (fix quickly)

---

## End of Day Routine (10 minutes)

### Step 1: Sales Check (3 minutes)
**Review today's sales**:
1. Go to **Sales** module
2. Filter by **Today's Date**
3. Count total sales
4. Check total amount

**✅ Verify**:
- All sales are finalized (not draft)
- Sales amounts look correct
- No duplicate sales

**⚠️ If Issues**:
- Draft sales → Ask sales staff to finalize
- Wrong amounts → Check with sales staff
- Duplicates → Delete duplicates

---

### Step 2: Production Check (3 minutes)
**Review today's production**:
1. Go to **Production** or **Studio** module
2. Check steps updated today
3. Check steps completed today
4. Check costs entered today

**✅ Verify**:
- Steps are being updated regularly
- Completed steps have costs entered
- Production orders are progressing

**⚠️ If Issues**:
- No updates → Check with workers
- Missing costs → Enter costs
- Stuck orders → Check what's blocking

---

### Step 3: Cash/Payment Check (2 minutes)
**Review payments**:
1. Go to **Sales** module
2. Filter by **Payment Status = Due**
3. Check total due amount
4. Review which customers owe money

**✅ Verify**:
- Due amounts are reasonable
- Old dues are being collected
- Payments are being recorded

**⚠️ If Issues**:
- High dues → Plan collection
- Missing payments → Check recording
- Old dues → Follow up with customers

---

### Step 4: Final Review (2 minutes)
**Quick checklist**:
- [ ] All sales finalized
- [ ] Production steps updated
- [ ] Costs entered
- [ ] No errors
- [ ] Team is comfortable

**✅ If All Good**:
- System is running well
- Business is on track

**⚠️ If Issues**:
- Note issues
- Plan fixes for tomorrow
- Communicate with team

---

## Daily Numbers That Matter

### Must Check Daily
1. **Sales Count**: Should be normal (not zero, not too low)
2. **Sales Amount**: Should match expectations
3. **Production Steps Updated**: Should see progress
4. **Payments Received**: Should match sales

### Must Check Weekly
1. **Total Sales for Week**: Compare with previous week
2. **Production Orders Completed**: Should be increasing
3. **Total Costs**: Should be tracked
4. **Due Amounts**: Should be collected

---

## Quick Action Guide

### If Sales Are Zero
1. Check if sales staff is working
2. Check if they can login
3. Check if they know how to create sales
4. Provide support if needed

### If Production Not Updating
1. Check if workers are using mobile app
2. Check if they can login
3. Check if steps are assigned
4. Provide support if needed

### If Errors Appear
1. Read error message
2. Note what was happening when error occurred
3. Contact support if needed
4. Don't ignore errors

### If System is Slow
1. Check internet connection
2. Refresh page/app
3. If still slow, contact support
4. Don't wait too long
