# Contacts Module Documentation

## Overview
Manages customer profiles, contact details, and interaction history.

## Components
- `src/app/components/contacts/ContactList.tsx`: Searchable list of all contacts.
- `src/app/components/contacts/ViewContactProfile.tsx`: Detailed profile view.
- `src/app/components/contacts/ContactLedgerDrawer.tsx`: Financial history of the contact.
- `src/app/components/contacts/QuickAddContactModal.tsx`: Fast contact creation.

## Recent Changes
- Standardized to dark mode.

## Tech Stack
- React
- Tailwind CSS
