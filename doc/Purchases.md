# Purchases Module Documentation

## Overview
Manages outgoing payments, purchase orders, and vendor transactions.

## Components
- `src/app/components/purchases/AddPaymentModal.tsx`: Modal for recording payments.
- `src/app/components/purchases/PurchaseList.tsx`: List of purchase records.
- `src/app/components/purchases/PurchaseDashboard.tsx`: Analytics for purchases.

## Recent Changes
- Manual edits have been made to `AddPaymentModal.tsx` to support specific payment accounts (Cash, Bank Transfer, JazzCash/EasyPaisa).

## Tech Stack
- React
- Tailwind CSS
- Lucide React Icons

## Cursor AI Instructions
When working on this module:
1. Preserve recent changes in `AddPaymentModal.tsx`.
2. Ensure financial calculations are accurate.
