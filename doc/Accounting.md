# Accounting Module Documentation

## Overview
Manages the financial backbone of the ERP, including accounts, ledgers, and funds transfers.

## Components
- `src/app/components/accounting/AccountingDashboard.tsx`: Financial overview.
- `src/app/components/accounting/AddAccountDrawer.tsx`: Adding new chart of accounts entries.
- `src/app/components/accounting/FundsTransferModal.tsx`: Moving money between accounts (e.g., Cash to Bank).

## Recent Changes
- Dark mode integration.

## Tech Stack
- React
- Tailwind CSS

## Cursor AI Instructions
When working on this module:
1. Double-entry bookkeeping principles should be observed where applicable.
2. Forms for money transfer must have validation to prevent errors.
3. UI must be clean and professional (`#111827` background).
