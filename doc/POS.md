# POS (Point of Sale) Module Documentation

## Overview
The POS system is designed for quick and efficient processing of sales and rentals. It has been recently redesigned to be modern and user-friendly.

## Components
- `src/app/components/pos/POS.tsx`: The main Point of Sale interface.

## Recent Changes
- Completely redesigned for a modern, visually appealing look.
- Improved usability and workflow.
- Strict dark mode (`#111827`) implementation.

## Tech Stack

- Tailwind CSS
